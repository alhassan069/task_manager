NLP-Powered Task Manager - Product Requirements Document

1. Product Overview

A task management web application that allows users to add and manage tasks using natural language input. Tasks are parsed using GPT-4-turbo to extract key components like task name, assignee, due date/time, and priority.

3. Key Features and Requirements

Feature Description Priority
